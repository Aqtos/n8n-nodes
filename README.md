# n8n-nodes-aqtos

This is an n8n community node. It lets you use Aqtos in your n8n workflows.

[n8n](https://n8n.io/) is a [fair-code licensed](https://docs.n8n.io/reference/license/) workflow automation platform.

## Installation

Follow the [installation guide](https://docs.n8n.io/integrations/community-nodes/installation/) in the n8n community nodes documentation.

## Operations

This node supports the following operations:

### Contact
- **Create** - Create a new contact (person or company)

### Expense
- **Create** - Create a new expense

### Invoice
- **Create** - Create a new invoice

### Lead
- **Create** - Create a new lead

### Task
- **Create** - Create a new task

## Credentials

To use this node, you need to authenticate with your Aqtos instance. You will need:

1. **Instance Name** - Your Aqtos instance name (e.g., "mycompany" for mycompany.aqtos.io)
2. **API Key** - Your Aqtos API key

### Getting Your API Key

1. Log in to your Aqtos instance
2. Navigate to your account settings
3. Generate your API key
4. Copy the API key and paste it into the credentials in n8n

## Compatibility

- Minimum n8n version: 1.0.0
- Tested with n8n versions: 1.0.0+
